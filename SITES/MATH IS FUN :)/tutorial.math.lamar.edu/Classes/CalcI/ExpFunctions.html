Section 1-7 : Exponential Functions

In this section were going to review one of the more common functions in both calculus and the sciences. However, before getting to this function lets take a much more general approach to things.

Lets start with \(b > 0\), \(b \ne 1\). An exponential function is then a function in the form,

\[f\left( x \right) = {b^x}\]

Note that we avoid \(b = 1\) because that would give the constant function, \(f\left( x \right) = 1\). We avoid \(b = 0\) since this would also give a constant function and we avoid negative values of \(b\) for the following reason.

Lets, for a second, suppose that we did allow \(b\) to be negative and look at the following function.

\[g\left( x \right) = {\left( { - 4} \right)^x}\]

Lets do some evaluation.

\[g\left( 2 \right) = {\left( { - 4} \right)^2} = 16\hspace{0.25in}\hspace{0.25in}g\left( {\frac{1}{2}} \right) = {\left( { - 4} \right)^{\frac{1}{2}}} = \sqrt { - 4} = 2i\]

So, for some values of \(x\) we will get real numbers and for other values of \(x\) we will get complex numbers. We want to avoid this so if we require \(b > 0\) this will not be a problem.

Lets take a look at a couple of exponential functions.

Example 1 Sketch the graph of \(f\left( x \right) = {2^x}\) and \(\displaystyle g\left( x \right) = {\left( {\frac{1}{2}} \right)^x}\).
Show Solution

Lets first get a table of values for these two functions.

\(x\) \(f(x)\) \(g(x)\)
-2 \(f\left( { - 2} \right) = {2^{ - 2}} = \frac{1}{4}\) \(g\left( { - 2} \right) = {\left( {\frac{1}{2}} \right)^{ - 2}} = 4\)
-1 \(f\left( { - 1} \right) = {2^{ - 1}} = \frac{1}{2}\) \(g\left( { - 1} \right) = {\left( {\frac{1}{2}} \right)^{ - 1}} = 2\)
0 \(f\left( 0 \right) = {2^0} = 1\) \(g\left( 0 \right) = {\left( {\frac{1}{2}} \right)^0} = 1\)
1 \(f\left( 1 \right) = 2\) \(g\left( 1 \right) = \frac{1}{2}\)
2 \(f\left( 2 \right) = 4\) \(g\left( 2 \right) = \frac{1}{4}\)

Heres the sketch of both of these functions.

This graph illustrates some very nice properties about exponential functions in general.

Properties of \(f\left( x \right) = {b^x}\)

  1. \(f\left( 0 \right) = 1\). The function will always take the value of 1 at \(x = 0\).
  2. \(f\left( x \right) \ne 0\). An exponential function will never be zero.
  3. \(f\left( x \right) > 0\). An exponential function is always positive.
  4. The previous two properties can be summarized by saying that the range of an exponential function is\(\left( {0,\infty } \right)\).
  5. The domain of an exponential function is\(\left( { - \infty ,\infty } \right)\). In other words, you can plug every \(x\) into an exponential function.
  6. If \(0 < b < 1\) then,
    1. \(f\left( x \right) \to 0{\mbox{ as }}x \to \infty \)
    2. \(f\left( x \right) \to \infty {\mbox{ as }}x \to - \infty \)
  7. If \(b > 1\) then,
    1. \(f\left( x \right) \to \infty {\mbox{ as }}x \to \infty \)
    2. \(f\left( x \right) \to 0{\mbox{ as }}x \to - \infty \)

These will all be very useful properties to recall at times as we move throughout this course (and later Calculus courses for that matter).

There is a very important exponential function that arises naturally in many places. This function is called the natural exponential function. However, for most people, this is simply the exponential function.

Definition : The natural exponential function is \(f\left( x \right) = {{\bf{e}}^x}\) where, \({\bf{e}} = 2.71828182845905 \ldots \).

So, since \({\bf{e}} > 1\) we also know that \({{\bf{e}}^x} \to \infty {\mbox{ as }}x \to \infty \) and \({{\bf{e}}^x} \to 0 {\mbox{ as }}x \to - \infty \).

Lets take a quick look at an example.
